## 📊 PRESENTATION STRUCTURE

### Act 1: THE INTERRUPTION (5 min)
**Goal**: Establish the 60-year UI paradigm just ended

**Slides 1-9**:
- IBM quote: "First new UI paradigm in 60 years"
- Cognitive real estate (not screen real estate)
- Everyone Tax (500 - 20 = 480)
- Application boundaries break thinking
- Vision: Three transformations

**Key Phrase**: "Everyone pays rent on everyone else's features"

---

### Act 2: THE PARADIGM (12 min)
**Goal**: Define ephemeral, intent-driven interfaces

**Slides 10-29**:
- PRIMARY CONCEPT: "Your intent is the layout"
- Three characteristics (purpose-driven, context-aware, persistent underpinnings)
- Code example (purpose vs timer)
- Why Now: Generative AI leap
- Conversation as state machine
- Context demo: Schedule (morning/midday/evening)
- **LIVE DEMO**: Travel planner (Flutter + Gemini)

**Key Phrase**: "The conversation history becomes the state machine"

---

### Act 3: THE SHIFT (8 min)
**Goal**: Explain paradigm implications

**Slides 30-39**:
- Application-centric → Intent-centric
- Don't navigate to features, features materialize
- Workflow comparison (5 apps vs 0 context switches)
- Persistence paradox: What dissolves vs persists
- "Dissolution isn't erasure. It's integration."
- Design principles: Transparency, override, explainability
- Cultural shift: Design for movement, not monuments

**Key Phrase**: "Don't navigate to features - features materialize around you"

---

### Act 4: THE PATH FORWARD (5 min)
**Goal**: Enable action with concrete next steps

**Slides 40-49**:
- What exists today (building blocks)
- Timeline: TODAY / 1-2yr / 3-5yr
- Honest assessment: "Simple ephemeral interfaces today, full paradigm evolving"
- Why act now: Fraunhofer quote
- Two paths: Lead now or catch up later
- **FIRST STEP**: "One screen. One context signal. Two states."
- First sprint: 5 steps, 2 weeks
- Closing: "60 years: static... / Now: breathing..."
- **CALL TO ACTION**: "Are you ready to build it?"

**Key Phrase**: "One screen. One context signal. Two states."

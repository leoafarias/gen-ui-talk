## Usage Guidelines

### When Creating Slides:
1. Use `master_outline_30min.md` as the blueprint
2. Reference `phase3_slide_manifest.md` for visual specs
3. Check `phase2_terminology_standard.md` for correct terms

### When Reviewing Content:
1. Verify against `phase2_core_concepts_final.md`
2. Check memorable phrases in `phase2_quotes_placement.md`
3. Validate structure against `phase3_slide_manifest.md`

### When Making Changes:
1. Update `master_outline_30min.md` first
2. Document in appropriate phase file
3. Re-validate against CLAUDE.md principles

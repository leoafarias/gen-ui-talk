## PRESENTATION OVERVIEW

**Core Message**: We've been building application-centric UIs—you navigate to features through static screens. Now we're building intent-centric UIs—ephemeral interfaces that compose around what you're trying to do, shaped by context, dissolved when done.

**Presentation Flow**:
- Act 1 (5 min): Establish problem - static interrupts cognition
- Act 2 (12 min): Present solution - ephemeral, intent-driven interfaces
